## 1. 시스템 철학

### 1-1. 핵심 비유
- **나침반(매크로)**: 장 시작 전 오늘 하루의 방향(시장 레짐) 결정
- **레이더(1분 데이터)**: 매분 실시간으로 수급·옵션·기술 신호 감지
- **자가학습**: 틀린 예보를 다음 분에 즉시 반영하는 바둑 AI

### 1-2. 절대 원칙
```

→ 15:10 강제 청산, 예외 없음
→ 수익 중이어도, 손실 중이어도
→ 오버나이트 포지션 절대 금지
이유: 1분봉 시스템은 야간에 데이터가 없고,

## 2. 전체 실행 흐름

```

       ↓

STEP 1: 과거 예측 검증 (T-1분→1m결과, T-3분→3m결과 ... 실시간 채점)
STEP 2: SGD 온라인 자가학습 (검증된 샘플로 즉시 모델 업데이트)
STEP 3: GBM 배치 재학습 (30분마다, 오늘 누적 데이터 사용)
STEP 4: 피처 생성 (수급·옵션·기술·매크로 통합)

STEP 8: 청산 트리거 감시 (우선순위 1~4순위)
STEP 9: 예측 DB 저장
       ↓
15:10  강제 청산 — 전 포지션 청산
15:40  자가학습 일일 마감
       (실제결과 업데이트 → 정확도 리포트 → SHAP 피처 심사)

## 9. 청산 관리

### 9-1. 당일 청산 절대원칙

```
15:10 → 강제 청산 시작
15:18 → 최종 마감 (슬리피지 감안)
예외 없음. 수익 중이어도, 손실 중이어도.
```

### 9-2. 청산 우선순위

| 순위 | 트리거 | 액션 |
|------|--------|------|
| 1순위 (즉시) | 하드 스톱 초과 / 앙상블 반대 신호 / CVD+VWAP 동시 역전 | 전량 즉시 청산 |
| 2순위 (다음봉) | SHAP 피처 2개 붕괴 / 옵션 플로우 역전 / 트레일링 스톱 | 전량 다음봉 청산 |
| 3순위 (부분) | 1·2·3차 목표가 도달 | 부분 청산 (33%·33%·34%) |
| 4순위 (강제) | 15:10 시간 도달 | 잔량 전량 청산 |

### 9-3. 손절 기준

```
하드 스톱:     진입가 ± ATR × 1.5
구조적 손절:   직전 스윙 고점/저점 돌파
VWAP 손절:     VWAP 반대 방향 돌파 시 즉시
최대 허용 손실: 계좌의 1% (절대 금액)
```

### 9-4. 부분 청산 및 트레일링 스톱

```
1차 청산 (33계약): 수익 +1 ATR → 손절을 본전으로 이동
2차 청산 (33계약): 수익 +1.5 ATR → 손절을 +0.5 ATR로 이동
3차 청산 (34계약): 신호 반전 or 최대 보유 시간 도달

트레일링 스톱:
  수익 < 1 ATR   → 하드 스톱 유지
  수익 1~1.5 ATR → 본전 손절
  수익 1.5~2 ATR → +0.5 ATR 보장
  수익 > 2 ATR   → 최고점 - 1 ATR 추적
```

"""

5단계 호가 가중 + 시간 감쇠 적용

DEPTH_WEIGHTS = [0.50, 0.25, 0.13, 0.07, 0.05]   # 1~5단 가중치

"""
