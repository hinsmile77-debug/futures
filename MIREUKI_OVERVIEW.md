# 미륵이 프로젝트 개요

## 프로젝트 별칭
- **미륵이**(“Mireuk”), KOSPI 200 선물 1분 봉 데이터 기반 자동매매 엔진

## 핵심 목표
- 오버나이트 없이 수익 창출을 목표로, 안전장치와 데이터가 기반인 실행 파이프라인을 구현

## 주요 인프라
- **Python 3.7 (32‑bit)**, conda env `py37_32`
- **Windows 전용** Kiwoom OpenAPI+ COM/OCX 사용
- `SciPy 1.5.4 (32‑bit)`, `scikit‑learn 1.0.2` + `joblib 1.1.1`
- TR 코드 **OPT50029**(OPT10080은 4.0 API 제한으로 사용 금지)

```

```
