## 2. 채용 항목 — 4건

### 2-1. 멀티 타임프레임 분석 ★★★★★

**핵심 차이점**: "멀티 호라이즌"과 "멀티 타임프레임"은 다른 개념.

```
기존 (멀티 호라이즌):
  현재 시점에서 1분봉으로 → 1분 후, 3분 후, 5분 후 ... 예측

신규 (멀티 타임프레임):
  1분봉 + 5분봉 + 15분봉 캔들을 동시에 분석
  → 5분봉 추세에 부합하는 1분봉 신호만 채택
```

**채용 이유**: 1분봉 노이즈를 상위 타임프레임으로 필터링.
헤지펀드 표준 기법. 거짓 신호 -30% 기대.

**구현 모듈**:
```
features/technical/
├── multi_timeframe.py        # 5분봉·15분봉 추세 분석
└── htf_filter.py             # 상위 타임프레임 필터
```

**적용 로직**:
```
5분봉 추세 == "상승" and 15분봉 추세 == "상승":
  → 1분봉 매수 신호 가중치 ×1.3

5분봉 추세 == "하락":
  → 1분봉 매수 신호 차단
```

### 2-2. 시장 상태 분류 (미시 레짐) ★★★★

**기존 시스템과의 차이**: 매크로 레짐(RISK_ON/OFF)만 존재.
미시 레짐(추세장/횡보장/급변장)은 없음.

```
기존 매크로 레짐 (1일 1회):
  RISK_ON / NEUTRAL / RISK_OFF
  → 진입 기준 신뢰도 조정용

신규 미시 레짐 (매분):
  추세장(ADX>25) / 횡보장(ADX<20) / 급변장(ATR 급등)
  → 전략 자체를 바꿈
```

**채용 이유**: 추세장에서 역추세 전략은 손실, 횡보장에서 추세 전략은 손실.
시장 상태에 맞는 전략 선택은 수익률 핵심 변수.

**구현 모듈**:
```
collection/macro/
├── micro_regime.py           # ADX·ATR 기반 미시 레짐 분류
└── regime_strategy_map.py    # 레짐별 전략 매핑
```

**분류 로직**:
```
ADX > 25 and ATR < 평균1.5배:  "추세장" → 추세추종
ADX < 20 and ATR < 평균:        "횡보장" → 역추세 (개인 역발상)
ATR > 평균 2배:                  "급변장" → 거래 중단/사이즈 축소
나머지:                          "혼합"   → 표준 앙상블
```

v5 "레짐별 전용 모델"과 결합 시 시너지.

### 2-3. 시간대 전략 분리 ★★★

**기존 시스템과의 차이**: 금지 시간만 존재.
시간대별 전략 차별화 없음.

```
한국 KOSPI 선물 시간대별 특성:

09:05~10:30 — 변동성 최대, 외인 방향성 베팅 활발
  → 추세추종 우위, 신뢰도 기준 상향

10:30~11:50 — 안정 추세 형성기
  → 표준 앙상블, 모든 전략 적용

13:00~14:00 — 점심 후 유동성 회복
  → 외인 재진입 감지, 신호 가중

14:00~15:00 — 마감 지향 변동성
  → 추세 가속 또는 청산 압력 구간

15:00~15:10 — 청산 임박
  → 신규 진입 금지, 청산 우선
```

**채용 이유**: 같은 신호도 시간대에 따라 의미가 다름.
시간대별 신뢰도 가중치 조정만으로 정확도 +3% 기대.

**구현 모듈**:
```
strategy/entry/
└── time_strategy_router.py   # 시간대별 전략·가중치 조정
```

### 2-4. 분할 진입 (2단계, 등급별) ★★★

**기존 상태**: 분할 청산(33·33·34%)은 있으나 분할 진입은 없음.
한 번에 풀사이즈 진입.

**부분 채용 이유**:
```
1분봉 시스템에서는 부분 적용 권장
  - 풀 분할(3단계)은 신호 사라지기 전 진입 못 할 수 있음
  - 2단계 분할(50%+50%) 정도가 적합
```

**진입 등급별 진입 방식**:
```
A급 (체크리스트 6개 통과): 100% 즉시 진입 (강한 신호 놓치지 않기)
B급 (4~5개 통과):          50% 진입 → 1분 후 가격 확인 → 추가 50%
C급 (2~3개 통과):          50% 진입 → 손절 도달 시 추가 진입 안 함
```

## 3. 제외 항목 — 3건

### 3-1. 오더플로우 분석 — 이미 더 진보된 형태로 구현

```
보완 제안 (3.3): 매수/매도 체결 비율, 호가 잔량 변화, 체결 속도

기존 보유:
  - CVD 다이버전스 (CORE) — 누적 매수·매도 차이
  - OFI (CORE) — 호가 잔량 변화
  - LOBID — 5단계 호가 가중 (학술 검증)
  - Microprice — 호가 압력 정밀 측정 (헤지펀드 표준)
  - Tick Imbalance — 체결 비율

학술 검증된 진보된 지표가 이미 있으므로
보완 제안의 단순 비율 지표는 추가 가치 없음
```

### 3-2. EMA(9·21·50) — 1분봉에 부적합

```
EMA의 문제:
  1. 후행 지표 (가격 변화 후 신호 발생)
  2. 1분봉 노이즈 환경에서 거짓 신호 다발
  3. 우월한 추세 지표 이미 보유
     - AMA (적응형 모멘텀, ER 자동 조정)
     - VWAP (기관 기준선)
     - LOBID (호가 압력)

EMA 골든크로스 같은 전략은 1분봉에서 거의 작동하지 않음
추가 가치 없음 → 제외
```

### 3-3. 리스크 관리 일반 — 이미 더 강력하게 구현됨

```
보완 제안 (3.5): 연속 손실 차단, 일일 최대 손실 제한, 변동성 급증 시 중단

기존 보유 (v5 설계 완료):
  - Circuit Breaker 5종 트리거
    1. 1분 내 신호 5번 반전
    2. 5분 내 손절 3번 연속
    3. 30분 정확도 < 35%
    4. 변동성 ATR 3배 초과
    5. API 응답 5초 초과
  - Kill Switch + Emergency Exit
  - 계좌 일일 한도 2%
  - 연속 손절 3회 자동 중단

이미 헤지펀드 수준 구현 → 추가 불필요
```
