### 기발한 고도화 아이디어

#### A. "SGD 헬스 게이트"
SGD 정확도가 50% 미만일 때 → 진입을 GBM-only 판단으로 전환.
```
IF sgd_accuracy_50bar < 50%:
    ensemble_blend = GBM_100%  # SGD 완전 배제
    block_new_entry = True     # 또는 최소한 size × 0.5
```
SGD가 노이즈를 주입하는 게 아니라, 아예 끄는 전략.

#### B. "방향 모멘텀 필터"
```
IF last_3_directions != all_same:
    block_new_entry = True
    reason = "방향 일관성 부족 (채찍장)"
```
오늘 T4/T5/T6 3연속 손실 전부 회피 가능.

#### C. "재시작 안전벨트"
```
IF seconds_since_restart < 300:
    block_new_entry = True
    existing_stop_multiplier = 2.0  # 갭 리스크 대비
```

#### D. "실시간 승률 기반 동적 Calibration"
```
actual_A_winrate = rolling_20_trades_winrate_A  # e.g. 0.44
calibrated_conf = raw_conf × (actual_A_winrate / 0.70)
```
Platt scaling을 넘어 실제 승률로 confidence 보정.

#### E. "SGD 가중치 변동률 기반 진입 타이밍"
```
IF delta_sgd_weight_5min > +10%:   # SGD 급상승 = 모델 신뢰도 상승
    entry_bonus = True             # 진입 조건 완화
IF delta_sgd_weight_5min < -10%:   # SGD 급락 = 모델 불안정
    block_entry = True
```
SGD 가중치 변화 속도가 시장 방향성의 선행 지표. 오늘 데이터로 backtest 시 승률 80%+ 예상.

#### F. "체크리스트-Confidence 교차 검증"
```
IF checklist_pass >= 7 AND ensemble_grade == "X":
    upgrade_to_grade = "B"  # 체크리스트 우선
```
T8 사례: conf=62.7%(B등급) + checklist 7/9 = A등급 승격 → 대박.
체크리스트가 raw confidence보다 실전 승률 예측력이 높을 가능성.
